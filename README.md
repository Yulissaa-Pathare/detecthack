# DetectHack

DetectHack is an AI-powered platform that detects manipulated images and videos and provides an explainable analysis report with timestamps and reasons.

## Live Demo

Frontend: https://detecthack.vercel.app/  
Backend API: https://detecthack.onrender.com/

## Features

- Deepfake detection for images and videos
- Spatial and temporal media analysis
- Confidence score for authenticity
- Detailed analysis report
- Timestamp-based anomaly detection

## Tech Stack

Frontend: HTML, CSS, JavaScript  
Backend: FastAPI (Python)  
AI Processing: OpenCV, PyTorch, NumPy  
Deployment: Vercel and Render
